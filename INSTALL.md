# How to install Sidekick manually

Sidekick is currently shared as a manual install beta through GitHub.

This means you need to load it manually in Chrome.

## Step 1: Download the ZIP

1. Go to the latest GitHub Release.
2. Download the Sidekick ZIP file.
3. Unzip the file on your computer.

## Step 2: Open the Chrome Extensions page

Open Chrome and go to:
chrome://extensions

Step 3: Turn on Developer Mode

On the Chrome Extensions page, turn on:

Developer mode

This is usually shown at the top-right.

Step 4: Load the extension

Click Load unpacked.
Select the unzipped Sidekick extension folder.
Click Open / Select Folder.

Step 5: Open a new tab

Open a new tab in Chrome.

Sidekick should now replace your new tab page.

Optional AI setup

Sidekick works without AI.

To use AI features:

Open the Sidekick extension popup.
Go to Settings.
Add your Groq API key.
Test the key.
Turn on AI Mode.
Important note

Because this is a manual beta, Chrome may show developer-mode warnings. This is expected for manually loaded extensions.

If anything does not work, please share feedback using the feedback form linked in the README.
