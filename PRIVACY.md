# Sidekick Privacy Note

Sidekick is designed as a privacy-first browser extension.

## Key privacy decisions

- No account is required.
- No analytics are used.
- No telemetry is collected.
- No remote config is used.
- User data is stored locally in the browser.
- AI Mode is off by default.
- AI features require the user to add their own Groq API key.
- The extension is designed to work even when AI is off.

## What data stays local

Sidekick stores product data locally, such as:

- Tasks
- Daily focus
- Workspaces
- Read Later items
- Settings
- Bookmark order
- Focus blocklist

## AI features

AI features are optional.

If the user enables AI Mode, selected data may be sent to the configured AI provider only when the user uses AI-powered features such as:

- Explain selected text
- Jargon explanation
- Ask across tabs
- AI Workspace Summary
- AI workspace name suggestion

## Why this matters

Browser tabs can contain sensitive personal or work context.

Sidekick is designed to be useful without requiring users to send browser data to AI.
