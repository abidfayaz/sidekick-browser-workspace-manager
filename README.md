# Sidekick — Privacy-first AI-assisted browser workspace manager

Sidekick is a Chrome extension that helps people reduce browser chaos, save work context, stay focused, and resume unfinished work later.

AI is optional. The product is designed to be useful even when AI is off.

## Why I built this

Browser tabs are not just clutter.

They often represent unfinished work, research, reminders, useful links, and context people are afraid to lose.

Sidekick explores a simple product idea:

**Detect tab chaos → Clean tabs → Save work → Resume later → Optional AI summary**

## What Sidekick can do

### Start work
- Daily focus
- Tasks
- Command search
- New tab dashboard

### Find work
- Open tab search
- History search
- Bookmark search
- Recently closed tabs

### Clean work
- Browser Health score
- Tab Triage
- Duplicate tab cleanup
- Review-before-closing safety for risky tabs

### Save work
- Save Workspace
- Read Later
- Bookmarks
- Screenshots
- Workspace notes

### Resume work
- Workspace list
- Resume Workspace
- Delete Workspace
- Optional AI Workspace Summary

### Stay focused
- Focus Mode
- Site blocking
- Temporary 5-minute pass

### Optional AI features
- Explain selected text
- Jargon explanation
- Ask across tabs
- Workspace name suggestion
- Workspace summary
- Triage suggestions

## Privacy-first approach

Sidekick is designed with privacy in mind:

- No account required
- No analytics
- No telemetry
- Data is stored locally
- AI Mode is off by default
- AI features require your own Groq API key
- The extension works without AI

## Optional AI setup

Sidekick works without AI.

To use AI features:

1. Create a Groq API key.
2. Open the Sidekick extension popup.
3. Go to Settings.
4. Paste your Groq key.
5. Test the key.
6. Turn on AI Mode.
